List Information

The ldm-users mailing list is an unmoderated list for announcements, discussions, bug reports, and questions about the LDM software and Unidata's Internet Data Distribution (IDD) system. It is also the main list for general data requests when data have been missed (other users will help replace the missing data). Typical postings include discussions on using the LDM; discussions of issues related to porting to various platforms; information about changes to stations; and discussion of issues and priorities affecting the further development of the LDM and the IDD.

Subscribing

In order to discuss the LDM and receive announcements about the LDM, you should join the ldm-users mailing list. To subscribe, send a blank email to: ldm-users-join@unidata.ucar.edu (be sure to respond to the confirmation email message sent from our mailing list manager to complete the subscription process).
