LDM Data-Product

An LDM data-product has two components: metadata and data.

Data-Product Metadata

The metadata component of a data-product has the following components:

  • Feedtype
  • Data-Product Origin
  • Data-Product Identifier
  • Data-Product Signature
  • Data-Product Creation-Time
  • Data-Product Size
  • Sequence Number

Data-Product Data

The data component of a data-product is completely arbitrary: it may be text, or binary data, or a mixture. It must have a finite size, however, and that size must be representable in the data-product size component of the associated data-product's metadata.
