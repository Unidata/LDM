How to Become the LDM user

There are two standard ways to become the LDM user -- depending on whether you are already logged onto the computer.

If you are not logged onto the computer, then simply logon as the LDM user.

If you are already logged onto the computer, then you can use the su utility to become the LDM user:

      su - ldm

The "-" option guarantees that the current working directory is the LDM user's home-directory and that the appropriate profile-file is read by the LDM user's shell in order to obtain the environment variable settings from the preinstallation steps.
