The LDM User

The LDM user is the user-account on the host computer under which the LDM is installed and operates. Typically, this user has the following characteristics:

  • The user's name is "ldm".
  • The user's home directory is "/opt/ldm". If you want to use another home-directory, then you should install the LDM system from a source distribution rather than from a binary one.
  • The subdirectory structure of the user's home directory is as specified in the relevant preinstallation step.
  • The user's system group is usually the same as that of the McIDAS and GEMPAK users so that the pqact process can create files in those user's data directories.
