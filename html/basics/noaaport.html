NOAAPORT Ingestion

When built appropriately, the LDM system can be used to receive and process data-products from a NOAAPORT receiver. In LDM releases prior to 6.10, this capability required the installation of a separate package called NOAAPORT. With release 6.10, the NOAAPORT package has been merged into the LDM package and only requires that the --with-noaaport option be given to the configure script in order to support this capability.

The documentation for NOAAPORT ingestion is distinct from the documentation for the rest of the LDM system because the NOAAPORT documentation was generated via the Doxygen system.
